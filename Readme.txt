This project is initiated by Lim Qi Jian
The project aims at creating a user-friendly Computational Electromagnetic tool
with the following features:
1. Simple CAD modelling
2. Importing IGES / STEP files
3. Meshing / Adaptive meshing
4. 1-D / 2-D / 3-D mesh
5. 1st / 2nd order elements
6. Inhomogeneous ODE solver
7. Eigenvalue equation solver
8. Radiation BC / PEC BC / Periodic BC
